================================================================================
                            DATA FLOW EXAMPLES
================================================================================

1. USER LOGIN FLOW:
   Client → CDN → API Gateway → Auth Service → PostgreSQL → Redis (Session)
                ↓
   Auth Service → JWT Token → Client

2. COURSE ENROLLMENT FLOW:
   Client → API Gateway → Course Service → PostgreSQL (Enrollment)
                ↓
   Course Service → Payment Service → Stripe API
                ↓
   Payment Service → Notification Service → SendGrid → User Email

3. VIDEO STREAMING FLOW:
   Client → CDN (CloudFront) → S3 Storage → Video Files
                ↓
   CDN → Client (Optimized Streaming - HLS/DASH)

4. MOCK INTERVIEW SCHEDULING FLOW:
   Client → API Gateway → Interview Service → PostgreSQL (Schedule)
                ↓
   Interview Service → Zoom/Google Meet API (Create Meeting)
                ↓
   Interview Service → Notification Service → SendGrid → Coach Email
                ↓
   Interview Service → Notification Service → SendGrid → Student Email

5. SEARCH FLOW:
   Client → API Gateway → Search Service → Elasticsearch
                ↓
   Elasticsearch → Results → Client

6. REAL-TIME NOTIFICATION FLOW:
   Service → Message Queue → Notification Service → SendGrid/Twilio
                ↓
   Notification Service → Socket.io → Client (Real-time Update)


================================================================================
                            SECURITY LAYERS
================================================================================

1. NETWORK LAYER:
   - VPC Isolation
   - Security Groups
   - WAF (Web Application Firewall)
   - DDoS Protection (CDN)

2. APPLICATION LAYER:
   - JWT Authentication
   - Role-Based Access Control (RBAC)
   - Input Validation & Sanitization
   - SQL Injection Prevention (ORM)
   - XSS Prevention

3. DATA LAYER:
   - Encryption at Rest (Database)
   - Encryption in Transit (TLS/SSL)
   - PII Data Encryption
   - Regular Backups
   - Database Access Controls

4. INFRASTRUCTURE:
   - Secrets Management
   - Certificate Management
   - Security Audits
   - Penetration Testing
   - Compliance (GDPR, SOC2)
