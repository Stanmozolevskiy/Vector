# Vector User Guide

## Getting Started

### Registration & Login
1. Register at `/register` with email and password
2. Verify your email via the link sent to your inbox
3. Log in at `/login`

### Profile
- Update your profile: Dashboard → Profile
- Upload a profile picture (JPG/PNG, max 5MB) — crop before saving
- Change password in Profile → Privacy tab

---

## Solving Problems

### Browse Questions
- Go to **Questions** to browse the interview question bank
- Filter by category, difficulty, or search
- Bookmark questions for later

### Solve a Question
1. Click a question to open it
2. Read the description, examples, and constraints
3. Write code in the editor (select language: Python, JavaScript, Java, C++, C#, Go)
4. Click **Run** to test against examples
5. Click **Submit** to validate against all test cases
6. Code auto-saves as you type

### View Solutions
- After solving (or to get hints), open the **Editorial** or **Solutions** tab
- Compare your approach with official solutions

---

## Mock Interviews

### Find a Peer
1. Go to **Find Peer Interview**
2. Set difficulty and category preferences
3. Click **Find Match** to be paired with another user

### Practice with a Friend
- Create a session and share the join link with your friend

### During the Interview
- Video chat and shared code editor
- Interviewer can change the question
- Timer shows remaining time
- Submit feedback when the session ends

## Progress & Gamification

### Dashboard
- View problems solved, mock interviews completed, and daily streak
- Track weekly goals

### Karma Points (Coins)
- Earn coins for: completing interviews, submitting feedback, publishing questions, profile completion
- View leaderboard at `/leaderboard`
- See how to earn at `/how-to-earn`
